QRmai 图像资源目录
==================

此目录存放模板图、皮肤等用户可上传的图像资源。

## 文件说明

| 文件名            | 用途                         | 来源     |
|------------------|-----------------------------|----------|
| p1.png           | 微信「生成二维码」按钮的截图    | 开发者   |
| p2.png           | 微信二维码消息区域的截图        | 开发者   |
| p1_user.png      | P1 模板（用户上传）            | 用户     |
| p2_user.png      | P2 模板（用户上传）            | 用户     |
| skin_1.png       | 皮肤图片（用户上传，编号递增）   | 用户     |
| skin_2.png       | 同上                         | 用户     |

## P1/P2 模板优先级

1. p1_user.png / p2_user.png（用户模板，最高优先级）
2. p1.png / p2.png（开发者默认模板）
3. config.json 中的 p1/p2 硬编码坐标（fallback）

## 使用方式

通过 Web 设置页面直接上传 P1/P2 模板和皮肤图片，无需手动操作文件。
